# Student–Teacher–Course Website Module

The objective is to create a complete Student–Teacher–Course management module with clearly separated responsibilities and dedicated pages.

---

## Student Page
- Create new students
- Read/View all students
- Update student details
- Delete students

---

## Teacher Page
- Create new teachers
- Read/View all teachers
- Update teacher details
- Delete teachers

---

## Courses Page
- Create new courses
- Read/View all courses
- Update courses
- Delete courses

---

## Course Assignments
Both assignments must happen through the course management flow:
- Assign one or more teachers to a course
- Assign one or more students to a course

---

## Enrolled Students Page
- Display all courses with assigned teachers
- Show enrolled students under each course

---

## Tech Stack

### Frontend
- MERN Stack  
- React.js  
- Axios  
- React Router DOM  

### Backend
- Node.js  
- Express.js  
- MongoDB  
- Mongoose  

---

## Important Notes
Each part of the project must strictly follow the “clear and distinct responsibility” guideline in the assignment.
